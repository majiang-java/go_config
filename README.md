# README #

这是一个golang版本的fromework-config的实现， 目前只实现了mysql

### 使用目标 ###

目标是为了使用golang的老师便利使用我们现有的配置中心架构

### 使用方法 ###

由于golang使用最先进的模块依赖策略，整个系统使用的包可以直接依赖网上的版本，但是如果不能上网，需要go get 相关软件
